# 《饿殍：明末千里行》×《哀鸿：城破十日记》大世界 · SillyTavern 角色卡

> **——明末炼狱里的救赎分支**

## 为什么做这张卡？
> 林翩翩太惨了。  
> 所以有了这张卡。

- **你可以**：在良出城的时候，顺手把穗儿她们救下来。
- **你也可以**：在林翩翩刚被卖掉的时候直接买下她，甚至提前从她母亲手里把人带走。
- **你还可以**：当一个纯粹的好人（或者更狠的恶人），在明末炼狱里走出另一条路。

至于《哀鸿：城破十日记》？  
**我没玩过。**  
我承认。哀鸿部分的信息全是从网上零散帖子、评论里推测出来的，肯定不准确。所以：

- **苏怜烟**（渐冻症）我没做，因为我没玩过续作。但你可以自己加，可以开挂救人。
- 其他《哀鸿》的角色、事件、细节，**欢迎玩过的人提交补充**。
- 也欢迎任何人补充更多的角色设定、分支剧情、随机事件——这是社区卡，不是我一个人说了算。

## 这张卡里有什么？

- **大世界框架**：明末崇祯年间，饥荒+兵灾+瘟疫，大地图可游荡。
- **可救赎角色**：林翩翩、满穗等。
- **自由开局**：你可以是流民、义军、商人、官员、江湖郎中……随便你编。

## 怎么用？

1. 装好 [SillyTavern](https://github.com/SillyTavern/SillyTavern)
2. 下载本仓库的 `epiao_aihong_world_card.md` 角色卡文件
3. 导入 → 开聊

## 我要怎么帮忙补充？

- 玩过《哀鸿》的：请提PR或开Issue，告诉我正确的角色、地名、事件。我直接抄进去。
- 有更好的救赎路线的：请写一段示例对话或事件脚本，我会合并进角色书。
- 觉得我写的某些设定太扯的：直接骂，然后告诉我对的应该是什么。
- **最简单的方法**：如果你会用 GitHub，可以 Fork 本仓库 → 修改 `.json` 文件 → 提交 Pull Request。我不会拒绝任何认真的补充。  
- 不会用 GitHub 也没关系：直接开 Issue 贴出你修改后的角色卡内容（或文字描述），我手动抄进去。
